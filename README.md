# book_app
Server-side templating with EJS

**Author**: Kent Sheats
**Version**: 1.5.0 

## Overview
This is a basic full-stack application for a book list which will include the ability to search the Google Books API, add books to a database, and then render those books from a PostgreSQL database. You will also add the ability to update the details of a book or remove it from the collection.
